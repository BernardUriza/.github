Philosophy in one paragraph

Every VAIR pipeline runs two competing models with different priors — Codex (OpenAI) generates the first draft; Claude (Anthropic) is given Codex's output as adversarial input and decides whether to extend, refine, or reject it. The contradiction between the two models is the load-bearing signal: agreement is high-confidence, disagreement triggers deeper analysis, rejection by Claude is logged as a finding in itself. Read-only pipelines (review) terminate at a verdict; write-capable pipelines (remedy, work) push Claude's final synthesis through Edit / Bash / Write tools and into git history. A substance gate refuses to push when zero changes were made, protecting against the "already implemented" false positive. When Codex is unreachable, every pipeline falls back to claude_only mode with byte-equivalent behavior — strict no-regression invariant.
